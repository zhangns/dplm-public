# DPLM

## Relevant paper

Emergent facilitation behavior in a distinguishable-particle lattice model of glass,

[Physical Review B 95, 184202 (2017)](http://dx.doi.org/10.1103/PhysRevB.95.184202).

This repo includes the simulation program for the Distinguishable-Particle
Lattice Model (DPLM) and scripts for data processing.
